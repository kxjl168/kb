然后又觉得只有一个小图片在上面不动不过瘾，于是给本站的喵小凡先森加上了尾巴摇动的动画，哈哈

 

这个动画使用了js,配合css实现，js动态增加css类型，css类型切换背景图片。

使用了4副小凡先生的图片

这个尾巴的动画差点玩死我了，像素级的编辑，还好能复制/旋转，搞了半天，起码尾巴甩起来还蛮飘逸的，HOHO

然后配合js动态修改css 

  Sprite.prototype.startAnimate = function() {

        var me = this;
        me.stopAnimate();
        $(me.default.sid).toggleClass("active");

        var index = 1;
        var isadd = true;

        me._timer = setInterval(function() {

            /*循环切换css,css中切换png图片 1234-》4321*/
            $(me.default.sid).toggleClass("o" + index);
            if (isadd)
                index++;
            else
                index--;
            if (index == 5) {
                isadd = false;
                index--;
            }
            if (index == 0) {
                isadd = true;
                index++;
            }

        }, 100);
    };
